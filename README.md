# App Cep Flutter

Aplicativo desenvolvido em Flutter para busca de CEPs com integração à API ViaCEP.

Funcionalidades
*Busca de CEPs com validação
*Exibe o endereço completo
*Integração com mapas
*Histórico de buscas




Tecnologias
*Flutter
*Dart
*MobX
*Hive
*API ViaCEP
